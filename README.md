# DiarySync (日记同步)
将日记站的每一篇文章都同步到handsome时光机，将一个typecho博客的文章同步到另一个typecho的某个页面下的评论区



## 能做什么？

分别两个typecho博客上安装DiarySyncReceiver&DiarySyncSender，经过配置后（配置教程在下文），在安装 Sender 的博客上您每发送一篇文章就会在安装 Receiver 的博客的对应页面下同步发送一则以 Sender 博客发送内容为内容的评论。

**即文章同步到评论** 

对我而言，它的功能就是将我的日记专用站的每一篇文章都同步到handsome的时光机内

## 使用场景

这个插件主要是为了配合我的 handsome 主题的时光机功能，对我而言，因为时光机的实现方式是通过评论并且增加页面模板实现，而我又专门建了一个日记站以记录日常，因此我很希望将日记站的每一篇内容都同步到时光机内，因此在AI的协助下，我一个晚上便完成了这个插件

# 如何使用

首先，下载插件，您将得到`` DiarySyncReceiver``和``DiarySyncSender``两个插件，其中将`DiarySyncReceiver`安装到主博客（有说说的博客），将`DiarySyncSender`安装到发送端博客（日记博客）

接下来进行配置：

对于`DiarySyncReceiver`：**通信密钥**请设置较强的通信密钥，**目标页面CID**是决定同步的文章评论在哪个页面下，CID可以通过后台编辑文章/独立页面时观察网址链接尾部的``...?cid=xxx``来得到，设置完毕后，日记博客的所有文章在点击发送后都会在主博客的对应文章/页面的评论区内同步发送

对于`DiarySyncSender`：**主博客 API 地址**请遵循例子中的写法，不要省略``/index.php/``和``https://``等内容，任何不规范的写法可能导致插件失效，（届时你可以取消Sender内关于日志的注释以查看问题所在）即**请不要填写会重定向的网页**；**通信密钥**请保证与主博客端一致，**发送者昵称 (选填)**与**发送者邮箱 (选填)**在不填写时会自动获取日记站的博主名称和邮箱



## 一些不足

本插件基本实现了博客与博客之间文章到评论的互通，但是依然存在不足，比如如果在日记博客中修改文章再次发送也会出发hook导致再次同步。以及仍有许多功能想要实现但未能实现，将在后期逐步完善插件，探寻更好的Handsome主题博客
